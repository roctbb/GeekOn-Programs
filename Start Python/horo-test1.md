### Проверочная работа по программированию

Напишите, что будет выведено в результате работы следующих программ. **Не забудьте подписать работу.** Удачи! :)

```python
print(2 + 4)
print(5 / 2)
print(2 ** 3)
print(10 // 3)
print(7 % 3)
```
